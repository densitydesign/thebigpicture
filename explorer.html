How to use

You can use the provided filters in order to navigate and compare the two datasets. The slider on the top controls the size of thumbnails, and the side button allows to alternate the actual picture with a square with the color resulting from the median value of the properties of each image. The source and categories buttons allows you to filter the dataset according to one or more sources or one or more categories. You can then sort the images according to their parameteres of Hue, Saturation, Brightness
